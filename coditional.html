If-else condition

Syntex 
    
Single line code


    if(condition)
        console.log("condition: True");
    else
        console.log("condition: False");

    
Multi-Line code
 

    if(condition){
        console.log("condition: True");
        console.log("condition: True");
    
    }

    else{
        console.log("condition: False");
    }

Switch-Case Condition

    Syntax Is 
    switch(condition){
        case 1:
           console.log("1");
           break;
        case 2:
           console.log("2"); 
           break;  
        default:
           console.log("Default Behaviour")   
    }
